Les fonctions

Les fonctions en C

Les fonctions en C sont des blocs de code qui peuvent être appelés pour effectuer une tâche spécifique. Elles permettent de diviser un programme en plusieurs parties plus petites et plus faciles à gérer. 

void say_hello() {
    printf("Hello, World!\n");
}

int main() {
    say_hello();
    return 0;
}

Cette fonction affiche le message Hello, World! à l'écran. Elle est ensuite appelée dans la fonction main pour être exécutée.

Les paramètres de fonction

Les fonctions peuvent prendre des paramètres en entrée pour effectuer des opérations spécifiques. Ces paramètres sont déclarés entre les parenthèses de la fonction. 

void greet(char name[]) {
    printf("Hello, %s!\n", name);
}

int main() {
    greet("Alice");
    greet("Bob");
    return 0;
}

Cette fonction prend un paramètre name de type chaîne de caractères et affiche un message de salutation personnalisé. Elle est ensuite appelée avec les noms Alice et Bob.

Les valeurs de retour

Les fonctions peuvent retourner une valeur à l'endroit où elles sont appelées. Cette valeur est définie par le type de retour de la fonction. 

int array_sum(int array[]) {
    int sum = 0;
    int size = sizeof(array) / sizeof(array[0]);

    for (int i = 0; i < size; i++) {
        sum += array[i];
    }

    return sum;
}

int main() {
    int numbers[] = {1, 2, 3, 4, 5};
    int total = array_sum(numbers);

    printf("The sum of the array is %d.\n", total);
    return 0;
}

Cette fonction calcule la somme des éléments d'un tableau d'entiers. Elle prend en paramètre le tableau numbers et retourne la somme totale.

Les prototypes de fonction

Les prototypes de fonction sont des déclarations qui permettent d'informer le compilateur de l'existence d'une fonction avant son appel. Ils sont généralement placés en haut du fichier source. 

void say_hello();
void greet(char name[]);
int array_sum(int array[]);

Ces prototypes déclarent les fonctions say_hello, greet et array_sum avant leur utilisation dans le programme.

Les prototypes de fonction

Dans l'exemple ci-dessous, la fonction add est définie après la fonction main.
En fonction de la version du compilateur, la compilation peut échouer avec un message d'erreur indiquant que la fonction add n'est pas déclarée.

Cependant, certains compilateurs peuvent autoriser cette pratique et ne pas détecter l'erreur lors de l'appel de la fonction add. 

#include <stdio.h>

// Déclarer le prototype de la fonction add ici

int main() {
    int result = add(3);
    printf("The sum is %d.\n", result);
    return 0;
}

int add(int a, int b) {
    return a + b;
}

Pour résoudre ce problème, il suffit de déclarer le prototype de la fonction add avant la fonction main.

Les fonctions récursives

Les fonctions récursives sont des fonctions qui s'appellent elles-mêmes pour résoudre un problème. Elles utilisent un cas de base pour terminer l'appel récursif. 

int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

int main() {
    int result = factorial(5);
    printf("The factorial of 5 is %d.\n", result);
    return 0;
}

Cette fonction calcule le factoriel d'un nombre en utilisant une approche récursive. Elle s'arrête lorsque le nombre atteint zéro.

Fonctions récursives

Les fonctions récursives peuvent permettre de remplacer des boucles dans certains cas. Elles peuvent être utilisées pour résoudre des problèmes mathématiques, des problèmes de recherche, des problèmes de tri, etc.
