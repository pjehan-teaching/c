Les conditions

Les conditions en C

Les conditions en C sont des structures de contrôle qui permettent d'exécuter un bloc d'instructions si une condition est vraie. 

int number = 42;

if (number == 42) {
    printf("The answer to the Ultimate Question of Life, the Universe, and Everything is 42.\n");
}

Cette condition va afficher le message si la variable number est égale à 42.

Imbrications

Il est possible d'imbriquer des conditions pour réaliser des tests plus complexes.

Par exemple, si nous souhaitons calculer un montant de réduction en fonction de l'âge et du nombre de produits achetés, nous pouvons écrire le code suivant :

Avec des conditions imbriquées

int age = 25;
int quantity = 3;
float discount = 0.0;

if (age < 18) {
    discount = 0.2;
} else if (quantity > 3) {
    if (age >= 18 && age < 65) {
        discount = 0.1;
    } else {
        discount = 0.15;
    }
}

Avec des conditions combinées

if (age < 18) {
    discount = 0.2;
} else if (quantity > 3 && (age >= 18 && age < 65)) {
    discount = 0.1;
} else if (quantity > 3) {
    discount = 0.15;
}
