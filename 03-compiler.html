Le compilateur C

Le compilateur C est un logiciel qui traduit le code source écrit en langage C en un code exécutable par la machine.

Il existe de nombreux compilateurs C, comme GCC, Clang, MSVC, etc.

Nous allons utiliser ensemble le compilateur GCC pour compiler nos programmes en langage C.

GCC est un compilateur libre et open source qui est compatible avec de nombreux systèmes d'exploitation (Linux, Windows, macOS, etc.) et permet de compiler des programmes en langage C dans sa version C99.

Installation de GCC

Pour installer GCC, nous allons utiliser un gestionnaire de paquets qui va télécharger et installer GCC et ses dépendances.

Windows

  1. Suivez les étapes d'installation sur le site scoop.sh. 
  2. scoop install gcc

MacOS

  1. Suivez les étapes d'installation sur le site brew.sh. 
  2. brew install gcc

Linux

sudo apt install gcc

Compilation d'un programme

Pour compiler un programme en langage C avec GCC, vous devez exécuter la commande suivante : 

gcc -std=c99 hello.c -o hello

Cette commande va compiler le fichier hello.c et générer un exécutable nommé hello.

Pour exécuter le programme, vous devez exécuter la commande suivante : 

./hello

Cette commande va exécuter le programme hello et afficher le message Hello, World!.

Étant donné que nous allons souvent compiler et exécuter des programmes en langage C, nous pouvons exécuter ces deux commandes en une seule fois : 

gcc -std=c99 hello.c -o hello && ./hello
