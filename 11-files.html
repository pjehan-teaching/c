Les fichiers

Les fichiers en C

Les fichiers en C sont des flux de données qui permettent de lire et d'écrire des informations sur un support de stockage. Ils sont utilisés pour stocker des données de manière permanente et pour communiquer avec des périphériques externes.

Voici un exemple de code qui écrit une chaîne de caractères dans un fichier : 

FILE *file = fopen("example.txt", "w");
if (file != NULL) {

    fprintf(file, "Hello, World!\n");
    fclose(file);
}

Cet exemple ouvre un fichier nommé example.txt en mode écriture (w), écrit la chaîne de caractères Hello, World! dans le fichier, puis ferme le fichier.

Les modes d'ouverture

Mode Description
r Ouvre un fichier en lecture seulement. Le fichier doit exister.
w Ouvre un fichier en écriture seulement. Crée un nouveau fichier ou écrase un fichier existant.
a Ouvre un fichier en écriture seulement. Ajoute des données à la fin du fichier ou crée un nouveau fichier.
r+ Ouvre un fichier en lecture et écriture. Le fichier doit exister.
w+ Ouvre un fichier en lecture et écriture. Crée un nouveau fichier ou écrase un fichier existant.
a+ Ouvre un fichier en lecture et écriture. Ajoute des données à la fin du fichier ou crée un nouveau fichier.

Les fonctions de gestion de fichiers

Fonction Description
fopen() Ouvre un fichier et retourne un pointeur de type FILE.
fclose() Ferme un fichier ouvert.
fgetc() Lit un caractère à partir d'un fichier.
fputc() Écrit un caractère dans un fichier.
fgets() Lit une chaîne de caractères à partir d'un fichier.
fputs() Écrit une chaîne de caractères dans un fichier.
fprintf() Écrit une chaîne de caractères formatée dans un fichier.

Ces fonctions permettent de manipuler des fichiers en C en effectuant des opérations de lecture et d'écriture.

Exemple de lecture de fichier

// main.c
#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "r");
    char buffer[255];

    if (file != NULL) {
        while (fgets(buffer, sizeof(buffer), file) != NULL) {
            printf("%s", buffer);
        }
        fclose(file);
    }
    return 0;
}
// example.txt
Hello, World!
This is an example file.
12345

Cet exemple lit le contenu du fichier example.txt ligne par ligne et l'affiche à l'écran. Le fichier contient trois lignes de texte.
