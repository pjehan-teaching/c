Le langage

Le langage C

Le langage C est un langage de programmation procédural créé en 1972 par Dennis Ritchie sur les bases du langage B créé par Ken Thompson.

Il est très utilisé dans le développement de systèmes d'exploitation et de logiciels embarqués.

Le langage C est un langage de bas niveau qui permet de programmer des algorithmes de manière efficace.

Ken Thompson et Dennis Ritchie

Ken Thompson et Dennis Ritchie ont contribué à la création du langage C et du système d'exploitation Unix.

Ken Thompson a également créé l'outil en ligne de commande grep, contribué à la création de l'encodage UTF-8 et participé à la création du langage Go.

Le langage C

Le langage C est à l'origine de nombreux autres langages de programmation, comme le C++, le C#, le Java, le JavaScript, le PHP, le Python, le Ruby, etc.

La langage C est un langage de bas niveau ce qui lui permet d'être très performant, mais il peut également être plus difficile de réaliser de simples tâches (comme par exemple la manipulation de chaînes de caractères) qu'avec des langages de plus haut niveau.

La gestion de la mémoire est également plus complexe en C qu'avec des langages de plus haut niveau qui gèrent la mémoire automatiquement.

Versions

Le langage C a évolué au fil des années et plusieurs versions ont été publiées.

La version originale du langage C est la version K&R (Kernighan & Ritchie) publiée en 1978.

La version standard du langage C est la version C89 (ANSI C) publiée en 1989.

La version moderne du langage C est la version C99 publiée en 1999.

C99

Nous utiliserons la version C99 pour écrire nos programmes en langage C même si des versions plus récentes du langage C existent (C11, C18, etc.) car la version C99 est plus largement supportée par les compilateurs C.

Cette version apporte notamment de nouvelles fonctionnalités au langage C telles que :

  • Les commentaires sur une seule ligne avec //
  • Les variables déclarées n'importe où dans une fonction
  • Les boucles for avec des variables déclarées dans l'initialisation
  • Les variables locales dans les blocs
  • Les variables de type booléen avec stdbool.h
  • Les tableaux de taille variables avec int n = 10; int array[n];
  • Les fonctions avec un nombre variable d'arguments avec stdarg.h
  • ...
