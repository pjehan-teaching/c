Variables

Les variables en C

Les variables en C sont des emplacements de mémoire qui permettent de stocker des valeurs.

Chaque variable en C possède un type qui définit le format de la valeur stockée et un nom qui permet de la retrouver.

Voici quelques types de variables en C :

  • Les entiers (int) pour stocker des nombres entiers (positifs ou négatifs)
  • Les réels (float, double) pour stocker des nombres à virgule
  • Les caractères (char) pour stocker des caractères
  • Les booléens (bool) pour stocker des valeurs booléennes (vrai ou faux).
    Les booléens ont été ajoutés dans la norme C99 et sont définis dans la bibliothèque stdbool.h.
  • Les tableaux (int[], float[], char[], etc.) pour stocker plusieurs valeurs
  • Les pointeurs (int*, float*, char*, etc.) pour stocker des adresses mémoire
  • ...

Déclaration de variables

Pour déclarer une variable en C, il faut préciser son type suivi de son nom : 

int age;

Cette instruction déclare une variable de type int (entier) nommée age.

Il est également possible d'initialiser une variable lors de sa déclaration : 

int age = 42;

Cette instruction déclare une variable de type int nommée age et lui affecte la valeur 42.

Le langage C est fortement typé, ce qui signifie que chaque variable doit être déclarée avec un type spécifique et que ce type ne peut pas être modifié par la suite : 

int age = 42;
age = "42"; // Erreur de compilation
warning: assignment to ‘int’ from ‘char *’ makes integer from pointer without a cast

Constantes

Les constantes en C sont des valeurs qui ne peuvent pas être modifiées une fois définies.

Pour déclarer une constante en C, il faut utiliser le mot-clé const suivi du type de la constante et de son nom : 

const int AGE = 42;

Par convention, les noms des constantes en C sont écrits en majuscules pour les distinguer des variables.

Comme son nom l'indique, une constante ne peut pas être modifiée : 

const int AGE = 42;
AGE = 43; // Erreur de compilation
error: assignment of read-only variable ‘AGE’

Afficher une variable

Pour afficher un message en C, il faut utiliser la fonction printf qui permet d'afficher du texte. Cette fonction est définie dans la bibliothèque standard stdio.h : 

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Cette instruction affiche le message Hello, World! suivi d'un saut de ligne.

Il est également possible d'afficher une variable en utilisant un format spécifique : 

int age = 42;
printf("J'ai %d ans\n", age);

Afficher une variable - Liste des formats

Voici quelques formats de variables couramment utilisés en C :

  • %d pour afficher un entier (int)
  • %f pour afficher un réel (float)
  • %lf pour afficher un double (double)
  • %c pour afficher un caractère
  • %s pour afficher une chaîne de caractères
  • %p pour afficher un pointeur
  • ...

Afficher une variable - Séquences d'échappement

En C, il est possible d'utiliser des séquences d'échappement pour afficher des caractères spéciaux :

  • \n pour un saut de ligne
  • \t pour une tabulation
  • \\ pour un antislash
  • \" pour un guillemet
  • \' pour une apostrophe
  • ...

Le caractère antislash \ (backslash en anglais) est utilisé dans de nombreux langages de programmation pour introduire des séquences d'échappement.

Conversion de types (casting)

En C, il est possible de convertir un type de variable en un autre type de variable en utilisant le casting.

Pour convertir une variable en un autre type, il suffit de préciser le type de conversion entre parenthèses avant la variable à convertir : 

float grade = 18.5;
int score = (int) grade; // 18

Cette instruction convertit la variable grade de type float en un entier int.

Dans cet exemple, la conversion est faite explicitement en précisant le type de conversion entre parenthèses, mais elle peut également être faite implicitement par le compilateur.

Saisi utilisateur

Pour lire une valeur saisie par l'utilisateur en C, il faut utiliser la fonction scanf qui permet de lire une valeur depuis l'entrée standard (clavier). Cette fonction est définie dans la bibliothèque standard stdio.h : 

#include <stdio.h>

int main() {
    int age;
    printf("Quel âge avez-vous ? ");
    scanf("%d", &age);
    printf("Vous avez %d ans\n", age);
    return 0;
}

Cette instruction demande à l'utilisateur de saisir son âge, lit la valeur saisie et l'affiche à l'écran.

Il est important de préciser l'adresse mémoire de la variable à lire en utilisant l'opérateur &. Nous reviendrons sur cette notion d'adresse mémoire lorsque nous aborderons les pointeurs en C.

Calculs

En C, il est possible d'effectuer des calculs en utilisant les opérateurs arithmétiques suivants :

  • + pour l'addition
  • - pour la soustraction
  • * pour la multiplication
  • / pour la division
  • % pour le modulo (reste de la division)

Il est également possible de combiner ces opérateurs pour effectuer des calculs plus complexes : 

int sum = 1 + 2; // 3
int difference = 5 - 3; // 2
int product = 2 * 3; // 6
int division = 10 / 3; // 3
int remainder = 10 % 3; // 1

Exercice - Circonférence d'un cercle

  • Écrire un programme en langage C qui demande à l'utilisateur de saisir le rayon d'un cercle et qui affiche la circonférence de ce cercle.
  • La formule pour calculer la circonférence d'un cercle est la suivante : C = 2 * π * r où C est la circonférence, π est le nombre pi (3.14159) et r est le rayon du cercle.
  • Exemple d'exécution : 
    Quel est le rayon du cercle ? 5
La circonférence du cercle est 31.4159

Exercice - Conversion unités de mesure

  • Écrire un programme en langage C qui demande à l'utilisateur de saisir une longueur en mètres et qui affiche cette longueur en kilomètres, en centimètres et en millimètres.
  • Exemple d'exécution : 
    Quelle est la longueur en mètres ? 1000
1000 mètres équivaut à 1 kilomètre, 100000 centimètres et 1000000 millimètres.

Incrémentation

L'incrémentation est une opération qui consiste à augmenter la valeur d'une variable de 1.

En C, il est possible d'incrémenter une variable en utilisant l'opérateur ++ : 

int count = 0;
count++; // 1

Cette opération est équivalente à l'addition de 1 à la variable : 

count = count + 1; // 1

Il est également possible d'incrémenter une variable d'une valeur spécifique : 

count += 10; // 10

Enfin, il est possible de décrémenter une variable en utilisant l'opérateur -- : 

count--; // -1

Les tableaux

Les tableaux en C sont des collections de variables du même type qui sont stockées de manière contiguë en mémoire.

Pour déclarer un tableau en C, il faut préciser son type, son nom et sa taille : 

int scores[5];

Manipulation des tableaux

Pour accéder à un élément d'un tableau en C, il faut utiliser son indice (position) qui commence à zéro : 

int scores[5] = {10, 20, 30, 40, 50};
int first = scores[0]; // 10
int last = scores[4]; // 50

Il est également possible de modifier un élément d'un tableau en C en utilisant son indice : 

int scores[5] = {10, 20, 30, 40, 50};
scores[0] = 100; // {100, 20, 30, 40, 50}

Chaînes de caractères

Les chaînes de caractères en C sont des tableaux de caractères terminés par un caractère spécial appelé caractère nul ('\0').

Pour déclarer une chaîne de caractères en C, il faut préciser son type et son nom : 

char message[6] = "Hello";

Cette instruction déclare une chaîne de caractères de taille 6 (5 caractères + 1 caractère nul) nommée message et lui affecte la valeur Hello.

Il serait également possible de déclarer chaque caractère individuellement : 

char message[6] = {'H', 'e', 'l', 'l', 'o', '\0'};

Cependant, il est également possible de déclarer une chaîne de caractères sans préciser sa taille : 

char message[] = "Hello";
' est utilisé pour définir un caractère et " une chaîne de caractères.

Chaînes de caractères

#include <stdio.h>

int main() {
    char message[] = "Hello";

    printf("%s\n", message); // Hello
    printf("%x\n", message[5]); // 0

    return 0;
}

Cependant, on ne peut pas déclarer une chaîne de caractères sans préciser sa taille et lui affecter une valeur : 

char message[]; // Erreur de compilation

Chaînes de caractères - concaténation

#include <stdio.h>

int main() {
    char message1[] = "Hello";
    char message2[] = "World";
    char message3[14];

    sprintf(message3, "%s, %s!", message1, message2);

    printf("%s\n", message3); // Hello, World!

    return 0;
}

La fonction sprintf permet de concaténer deux chaînes de caractères dans une troisième chaîne.

Il serait également possible d'utiliser la fonction strcat de la librairie string.h pour concaténer deux chaînes de caractères.

Exercice - Convertisseur de température

  1. Créer un programme en langage C qui demande à l'utilisateur de saisir une température en degrés Celsius.
  2. Convertir cette température en degrés Fahrenheit.
    La formule pour convertir une température de degrés Celsius en degrés Fahrenheit est la suivante : F = C * 9/5 + 32 où F est la température en degrés Fahrenheit et C est la température en degrés Celsius.
  3. Exemple d'exécution : 
    Quelle est la température en degrés Celsius ? 0
0 degrés Celsius équivaut à 32 degrés Fahrenheit.

Exercice - Aire d'un cercle

  1. Créer un programme en langage C qui demande à l'utilisateur de saisir le rayon d'un cercle.
  2. Calculer l'aire de ce cercle.
    La formule pour calculer l'aire d'un cercle est la suivante : A = π * r² où A est l'aire, π est le nombre pi (3.14159) et r est le rayon du cercle.
  3. Exemple d'exécution : 
    Quel est le rayon du cercle ? 5
L'aire du cercle est 78.5398

Exercice - ETA

  2. Calculer le temps de trajet en heures et en minutes.
  3. Exemple d'exécution : 
    Quelle est la distance en kilomètres ? 100
Quelle est la vitesse en kilomètres par heure ? 50
Le temps de trajet est de 2 heures et 0 minutes.
